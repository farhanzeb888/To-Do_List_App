To-Do List App
A simple and intuitive to-do list application to help you organize your daily tasks.

Features
Add new tasks

Mark tasks as complete

Delete tasks

Clear all tasks

Persistent task storage (using localStorage)

Responsive design

Technologies Used
HTML5

CSS3

JavaScript (ES6+)


How to Use
Clone the repository:

bash
git clone https://github.com/farhanzeb888/To-Do_List_App.git
Open index.html in your web browser

To add a task:

Type your task in the input field

Press Enter or click the "Add" button

To mark a task as complete:

Click on the task text

To delete a task:

Click the delete button (âœ–) next to the task

To clear all tasks:

Click the "Clear All" button
